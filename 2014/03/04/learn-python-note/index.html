python 简单入门教程的学习笔记

一直计划着学习了解一下python，前几天稍稍看了一下python相关的东西。

语言特点

在《python programming》中作者O’Reilly Media, Inc 强调了Python 四个概念：

Software quality
Developer productivity
Program portability
Component integration

软件质量:首先在python语言设计结构上面，会使软件质量提高，比如缩进，这一特性也让很多程序员抓狂。另外一方面，python 在不断地改善，python的社区很大。代码容易被理解。
开发效率:你很难在python中发现诸如类型声明、存储设计、内存管理等等东西，python优化了开发速度。有很多的开源库，可以调用c++,java。python类似于可执行的伪代码
程序可移植性:
组件交互:与其他语言的交互。control language and customization tool

另外适合用python编程的主要有一下几个部分：

System utilities
Internet scripting
GUI
component iteraction
Database access
Distributed programming
Rapid-prototying/development
Language-based modules
…

而python不适合的地方主要还是在有关速度方面，虽然有时候python的速度能够做到跟c++ 一样快，但是总的来说，是比不过那种语言的。

O’Reilly Media, Inc 也有讲到python也有它的弱点：

大量的调用，以及语言本身的一些特性，导致排错比较麻烦
有时候会导致你很难用python实现你自己的意图，可能写着写着，就没有按照你的意思做了。

语法特性

变量类型

看了一个[1] 中的教程，感觉python有点像javascript，不过我对javascript也不熟，相对于其他的诸如c++,c,java 等语言，python简单了很多。
没有了类型，声明一个变量只是如下操作：

a = 3
b = 4
print (a +b )
print (a ** b )   #这个是指数 a 的b 次方
print (a // b )   #floor division 去除余数，将结构只保留整数部分

这样就输出7,81,0了。 不知发现了没有，python中貌似没有分号

if

另外python使用缩进来取代花括号了。如if语句是这样的：

a = 1
if a > 2 :
    print(" a > 2" )
else :
    print(" a > 2 is false")
print("end")

输出的结果就是

a > 2 is false
end

while for

python中没有do while while 的使用如下
a = 1
while a <3
a+=1
print(“end” + a)

结果是：
end3

for 的使用如下:
a = [1,2,3]
for i in a
print(i)

函数的定义如下

def myFunction(a,b):
    return a + b
print(myFunction(1,2))

python中有三种数据结构

python中的三种数据结构是 dict（相当于map),list, tuples(不可修改的list，不能够做insert append ，delete 操作）

其实Strings 也是一个类。其中也包含了很多函数操作 python 中的数组序列是从0 开始的，但是它有-1 ，是指从最后一个元素起减1 ，另外python中的数组序列是包含起始位置，不包含结束位置的。如下面代码的输出
a = [1,2,3,4]
print(a[0:3]) # 结果会是[1,2,3]

python中用int ，str 强制类型转换，以及各式化，如：
print( ‘who are you %f’ % 123.33) #结果是 who are you 123.330000

python中的异常，读写文件和类

读写文件的例子如下：
f = open(“test.txt”, “r”) #opens file with name of “test.txt”
print(f.read(1))
print(f.read())
print(f.readline())
f.close()

结果类似于c++ 中的
f = open(“test.txt”,”w”) #opens file with name of “test.txt”
f.write(“I am a test file.”)
f.write(“Maybe someday, he will promote me to a real file.”)
f.write(“Man, I long to be a real file”)
f.write(“and hang out with all my new real file friends.”)
f.close()

类：

#ClassOne.py
class Calculator(object):
    #define class to simulate a simple calculator
    def __init__ (self):
        #start with zero
        self.current = 0
    def add(self, amount):
        #add number to current
        self.current += amount
    def getCurrent(self):
        return self.current

使用：
from ClassOne import * #get classes from ClassOne file
myBuddy = Calculator() # make myBuddy into a Calculator object
myBuddy.add(2) #use myBuddy’s new add method derived from the Calculator class
print(myBuddy.getCurrent()) #print myBuddy’s current instance variable
