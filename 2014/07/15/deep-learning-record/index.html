深度学习学习

Veröffentlicht am 2014-07-15

最近老师叫我了解一下深度学习的东西。看了一些东西，顺便总结一下，记录一下自己的想法。

深度学习是机器学习研究中的一个新的领域，其在语音识别、图像处理等方面已经取得了巨大成功。把学习结构看作一个网络，则深度学习的核心思路如下：

  1. 无监督学习用于每一层网络的pre-train;
  2. 每次用无监督学习只训练一层，将其训练结果作为其高一层的输入；
  3. 用自顶而下的监督算法去调整所有层

上面是深度学习的核心思想，刚开始接触深度学习的时候，完全就是看着那个RBM的经典demo，以至于思维完全限制在那些东西上面，后面接触多了，再回过头来看深度学习一些基本的介绍，思维才开阔起来，深度学习是一种学习方式，更多的应该是在于思想，就像上面那三个点。我开始看深度学习的时候，看到一篇论文，它是用卷积神经网络+SVM来进行人脸表情识别的。在我没明白了解深度学习时看的，那时候都没有明白，CNN其实本身就是有很多层的，对于卷积神经网络，在这篇文章中有详细的介绍：http://ibillxia.github.io/blog/2013/04/06/Convolutional-Neural-Networks/

pylearn2

深度学习有一个非常著名的库Theano，它是深度学习的Python库。pylearn2是基于Theano弄的一个库，它的源码在：https://github.com/lisa-lab/pylearn2,这个库中，提供了很多深度学习的基本算法。我最近基本就是使用这个库来做一些实验。而且这样弄出来的效果，能够快速的投入到使用中。

我使用的主要是https://github.com/lisa-lab/pylearn2/tree/master/pylearn2/scripts/icml_2013_wrepl/emotions这部分。

pylearn2使用了yaml格式文件，这种方式，让深度学习几乎就不用再去编码，直接修改yaml文件就好了。个人觉得非常好。下面是一个模型的例子：

 model: !obj:pylearn2.models.mlp.MLP {
    layers : [
        # To make this baseline simple to run, we use a very small and cheap convolutional
        # network. It only has one hidden layer, consisting of rectifier units with spatial
        # max pooling.
        !obj:pylearn2.models.mlp.ConvRectifiedLinear {
            layer_name: 'h0',
            kernel_shape: [8, 8],
            pool_shape: [4, 4],
            pool_stride: [2, 2],
            output_channels: 32,
            irange: .05,
            # Rather than using weight decay, we constrain the norms of the convolution kernels
            # to be at most this value
            max_kernel_norm: .9
        },

        !obj:pylearn2.models.mlp.Softmax {
            layer_name: 'y',
            # The classes are unbalanced. Set the bias parameters of the softmax regression
            # to make the model start with the right marginals. This should speed convergence
            # of the training algorithm.
            init_bias_target_marginals: *train,
            irange: .0,
            # There are seven different emotions to learn to recognize, i.e., 7 class labels
            n_classes: 7
        }
    ],
    # The inputs are 48x48 pixel images
    input_space: !obj:pylearn2.space.Conv2DSpace {
        shape: [48, 48],
        num_channels: 1
    }
},

一层一层的，并且输入空间也通过这种方式，直接进行描述，非常直观。上面就是一个两层的模型，一个MLP线性卷积层，一个Softmax。

上面只是一个非常简单的demo。我在使用的过程中，老师一直说应该要多一些层次，但是有的时候，如果数据量小的话，深度学习其实压根没有用。不过我尝试了通过加多层次，虽然测试集的准确率变低了，但是我使用其他的数据集作为测试集的时候，准确率还有显著的提高。说明增加多层，能够增加模型的稳定判断性。

使用总结

我不是一个理论方面的擅长者，但是做实验还是过得去的，而且深度学习本身也是无法解释的。我直接总结一下实验经验。

第一个就是上面说的，多层能够提高模型的对不同数据集的准确率。

深度学习需要在学习的过程中，自顶向下用监督算法学习每一层。所以在训练过程中，还需要指定一部分数据作为训练过程中的测试集。我一开始总是追求测试集跟训练集不一样，但是在小的数据集进行训练的时候，就发现结果非常不好，本身数据集就几百条数据，还抽取出一部分数据不用来训练，可能导致效果更加不佳，而且增加了训练结果的不确定性。所以我把整个数据集作为训练，然后抽取其中的一部分作为调整测试集。所以在训练过程中测试集其实也在训练集中使用过了。

不要光顾着使用多层，而忘了具体的数据，我一开始粗暴地使用了5层，结果发现经过5层的卷积后，特征直接就没了，压根就不够5层的，所以后面我在调小的每次卷积的大小，以使通过多层卷积后，还是能够得到一个合适的输出，用来作为最后的判断。

最后得到的结果也是相对不错的，虽然没有很高的准确率，但是平均5成的准确率还是能够勉强投入实用了。而且通过深度学习来判断，模型本身就相对于其他的更加可靠，预测出来的结果不会非常离谱。

####目前实验过程中一个最有效的模型
demo:

!obj:pylearn2.train.Train {
# Here we specify the dataset to train on. We train on only the first 25k of the examples, so
# that the rest may be used as a validation set.
# The "&train" syntax lets us refer back to this object as "*train" elsewhere in the yaml file
dataset: &train !obj:pylearn2.scripts.icml_2013_wrepl.emotions_yaxin.emotions_dataset.EmotionsDataset {
    which_set: 'train',
    start: 0,
    stop: 2593,
    # We preprocess the data with global contrast normalization
    preprocessor: &prepro !obj:pylearn2.datasets.preprocessing.GlobalContrastNormalization {
        sqrt_bias: 10,
        use_std: 1
        }
},
# Here we specify the model to train as being an MLP
model: !obj:pylearn2.models.mlp.MLP {
    layers : [
        # To make this baseline simple to run, we use a very small and cheap convolutional
        # network. It only has one hidden layer, consisting of rectifier units with spatial
        # max pooling.
        !obj:pylearn2.models.mlp.ConvRectifiedLinear {
            layer_name: 'h0',
            kernel_shape: [5, 5],
            pool_shape: [3, 3],
            pool_stride: [2, 2],
            output_channels: 64,
            irange: .05,
            # Rather than using weight decay, we constrain the norms of the convolution kernels
            # to be at most this value
            max_kernel_norm: .9
        },

        !obj:pylearn2.models.mlp.ConvRectifiedLinear {
            layer_name: 'h1',
            kernel_shape: [5, 5],
            pool_shape: [3, 3],
            pool_stride: [2, 2],
            output_channels: 64,
            irange: .05,
            # Rather than using weight decay, we constrain the norms of the convolution kernels
            # to be at most this value
            max_kernel_norm: .9
        },


        !obj:pylearn2.models.mlp.ConvRectifiedLinear {
            layer_name: 'h2',
            kernel_shape: [3, 3],
            pool_shape: [2, 2],
            pool_stride: [1, 1],
            output_channels: 32,
            irange: .05,
            # Rather than using weight decay, we constrain the norms of the convolution kernels
            # to be at most this value
            max_kernel_norm: .9
        },

        !obj:pylearn2.models.mlp.ConvRectifiedLinear {
            layer_name: 'h3',
            kernel_shape: [3, 3],
            pool_shape: [2, 2],
            pool_stride: [1, 1],
            output_channels: 32,
            irange: .05,
            # Rather than using weight decay, we constrain the norms of the convolution kernels
            # to be at most this value
            max_kernel_norm: .9
        },

        !obj:pylearn2.models.mlp.Softmax {
            layer_name: 'y',
            # The classes are unbalanced. Set the bias parameters of the softmax regression
            # to make the model start with the right marginals. This should speed convergence
            # of the training algorithm.
            init_bias_target_marginals: *train,
            irange: .0,
            # There are seven different emotions to learn to recognize, i.e., 7 class labels
            n_classes: 7
        }
    ],
    # The inputs are 48x48 pixel images
    input_space: !obj:pylearn2.space.Conv2DSpace {
        shape: [128, 128],
        num_channels: 1
    }
},
# We train using SGD and momentum
algorithm: !obj:pylearn2.training_algorithms.sgd.SGD {
        batch_size: 100,
        learning_rate: .001,
        learning_rule: !obj:pylearn2.training_algorithms.learning_rule.Momentum {
            init_momentum: .5,
        },
        # We monitor how well we're doing during training on a validation set
        monitoring_dataset:
            {
                'valid' : !obj:pylearn2.scripts.icml_2013_wrepl.emotions_yaxin.emotions_dataset.EmotionsDataset {
                    which_set: 'train',
                    start: 2000,
                    stop: 2608,
                    preprocessor: *prepro
                }
            },
        # We stop when validation set classification error hasn't decreased for 10 epochs
        termination_criterion: !obj:pylearn2.termination_criteria.MonitorBased {
            channel_name: "valid_y_misclass",
            prop_decrease: 0.,
            N: 50
        },
    },
    # We save the model whenever we improve on the validation set classification error
    extensions: [
        !obj:pylearn2.train_extensions.best_params.MonitorBasedSaveBest {
             channel_name: 'valid_y_misclass',
             save_path: "${PYLEARN2_TRAIN_FILE_FULL_STEM}_best.pkl"
        },
    ],
}
