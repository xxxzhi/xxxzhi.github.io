单次遍历随机选择算法

Veröffentlicht am 2013-11-25 | in 算法
单次遍历文件，从中选择m个记录，总记录不知道有多少

今天在对机器学习uci上面的数据进行处理的时候，需要对数据进行随机。由于数据是在文件里面的，而且又超大，如果一次性全部读取出来，然后。。。然后再然后，赶脚内存就消耗太大了。就想着，一次遍历，就能够选出想要的m个随机的数据。google 到了gocalf 的文章。在此表示感谢了，虽然我没完全看懂他说的。

自己想了好久毫无头绪，后面看了上面gocalf 的文章，感觉有些头绪了。但是他用python 写的，最主要是纠结在他那个证明中，不是很确认是否正确，不自己证明，感觉不妥。

算法过程

算法过程是这样的：

s1.初始化cur_index = 0,record_arr [m].
s2.从文件读取一个记录record，
s3.判断cur_index > m 如果为真 进行s4.如果为假则 record_arr[cur_index] = record.
s4.int ran = rand()%(cur_index+1);
    if(ran >= m) 跳到s5
    else record_arr[ran] = record;
s5.跳到s2.

证明

借用一下gocalf 里面的那个公式。额，公式编辑比较麻烦。

$$\begin{equation}
p[i]=\left{ \begin{array}
{r@{\quad: \quad}1}
1 \frac{m}{m+1} \frac{m+1}{m+2} …\frac{n-1}{n} & i<m \
\frac{m}{m+1} \frac{m+1}{m+2} *…\frac{n-1}{n} & i>=m \
\end{array} \right. \end{equation}$$

i>m 时，从m个后，才开始到该记录，选到该记录时，该记录有 $$\frac{m}{m+1}$$ 的机会被选中。然后在遍历到i+1时。i记录被选中后，有 $$\frac{m+1}{m+2}$$ 不被选中，也就是继续保留，持续下去，到了最后都没有被选中从已经选的record_arr 中挑出来。就算是最后被选中了。所以其概率就是所有的保留在record_arr 中的和。

i<=m 时，显然，一开始都会被选中放入到record_arr 中。这样一开始被选中的概率为1。之后不被挑出的概率也是
$$\frac{i}{i+1}$$ 。所以也是那个相乘。结果概率是一样的
$$\frac{m}{n}$$.证明相等。

代码

/*
 * SingleTraveralRandom.h
 *
 *  Created on: 2013年11月25日
 *      Author: houzhi
 */

#ifndef SINGLETRAVERALRANDOM_H_
#define SINGLETRAVERALRANDOM_H_

#include<time.h>
#include <stdlib.h>
#include <stdio.h>
#include <iostream>

using namespace std;

class SingleTraveralRandom {

public:
    SingleTraveralRandom(int select_nums);

    int SingleTraversal();

    void Init();

private:
    int Random(int max);


    size_t cur_index;

    //选择多少
    size_t select_nums;
};

SingleTraveralRandom::SingleTraveralRandom(int select_nums){
    this-> select_nums = select_nums;
    cur_index = 0;
}

inline int SingleTraveralRandom::Random(int max){
    srand((unsigned )time(NULL));

    int res = rand() % max;

    return res;
}

/**
 * 初始化
 */
inline void SingleTraveralRandom::Init(){
    cur_index = 0;
}

int SingleTraveralRandom::SingleTraversal(){
    cur_index ++;
    if(cur_index <= select_nums ){
        return cur_index-1;
    }else{
        int random = Random(cur_index);
        if(random <=select_nums){
            return random-1;
        }else{
            return -1;
        }
    }
}

#endif /* SINGLETRAVERALRANDOM_H_ */
